## v4.21 扩图 Outpainting（§6.1.5）· 图像编辑闭环补全

**状态**：✅ 已落地（2026-07-19）；对应 v4.20 结论「下一步 ① 扩图 Outpainting」。无限画布图像编辑三件套（图生图 / 局部重绘 / **扩图**）齐备，创作延展无界。

- **折叠分组**：参数卡片 / 选中节点信息 / LoRA+ControlNet 编辑区独立折叠（ArrowIcon 旋转指示），默认展开，可点击收拢节省纵向空间
- **Enter 键提交**：prompt `<textarea>` 监听 `onKeyDown`，Enter 直接触发生成（`!shiftKey` 避免 Shift+Enter 换行冲突）
- **错误自动消失**：error banner 渲染后启动 8s `setTimeout`（`useRef` 存句柄），新错误重置计时器，组件卸载清理定时器
- **模式标签优化**：`MODE_META` 扩展为 6 项（文生图绿 / 图生图蓝 / 局部重绘紫 / 扩图橙 / 文生视频青 / 图生视频粉），模式按钮渲染对应彩色标签

**2. App 层增强（App.tsx）**
- **Toast 通知**：全局 Toast 条，类型 `loading|success|error`，渲染 icon + 文字 + 渐入动画 + 3s 自动消失（`useEffect` + `setTimeout`）
- **状态自动刷新**：`/api/status` 每 3s `setInterval` 轮询（GPU 使用率/温度/VRAM），`useEffect` 清理 `clearInterval`
- **快捷键底栏**：画布底部固定半透明提示条（`Space+G` 生成 / `Del` 删除 / `Ctrl+Z/Y` 撤销重做 / `Scroll` 缩放），`opacity: 0.6`

**3. 深色主题统一（theme.ts）**
- **40+ 语义 token**：
  - 底色：`bg`(#1a1a2e) / `surface`(#16213e) / `surfaceHover`(#1a2744) / `surfaceActive`(#1e3050)
  - 边框：`border`(#2a3a5c) / `borderFocus`(#4a7cff) / `borderError`(#ff4a6a)
  - 文字：`textPrimary`(#e8e8f0) / `textSecondary`(#a0a8c0) / `textMuted`(#606878) / `textInverse`(#1a1a2e) / `textAccent`(#4a9eff)
  - 强调：`accent`(#4a7cff) / `accentHover`(#6a9aff) / `danger`(#ff4a6a) / `warning`(#ffb84a) / `success`(#4aff8b)
  - 图谱 9 色 + 模式 6 色 + 工作流 9 色（见迭代记录 v4.25 工作流着色体系）
- **替换范围**：约 150+ 处硬编码 `#hex` 色值在 `ControlPanel` / `Canvas` / `App` / `WorkflowGraph` / `WorkflowPanel` / `MaskEditor` / `store` 中全部替换为 `theme.xxx` 引用
- **收益**：全仓库色彩由 `theme.ts` 单一文件控制，后续换肤只需改一处
