## v4.23 — ControlNet 节点化

**时间**: 2026-07-19  
**类别**: Phase 1 增强  

### 落地能力
- **攻克关键难点**: `controlnet++_union_sdxl_promax` union 模型的 `type` 字段为 `COMBO`（无法静态枚举）→ 以运行时校验解决：构造假 checkpoint + 真模型提交 `/prompt`，捕获 `node_errors` 得权威合法集合
- **9 合法类型**: auto / openpose / depth / hed·pidi·scribble·ted / canny·lineart·anime_lineart·mlsd / normal / segment / tile / repaint
- **预处理器注册表**: `CONTROLNET_PREPROC` 单源映射（OpenposePreprocessor / DepthAnything / Scribble / Canny / BAE-NormalMap / OneFormer / Tile）
- **后端链**: 控制图独立 LoadImage → 预处理器 → ControlNetLoader → SetUnionControlNetType(type) [union] → ControlNetApply
- **前端**: 图片节点按钮拆「+LoRA」「+ControlNet」；控制面板补全模型/类型/强度下拉；`Canvas` 卡片副标题
- **测试**: pytest 38→44 passed（+6 ControlNet）；build 467 kB

### 里程碑
> 控制节点化 **完整闭环**（LoRA + ControlNet 双轨可拖拽连线）

**时间**: 2026-07-19  
**类别**: 大特性 · 文生视频 / 图生视频  

### 落地能力
- **后端蓝图**: `_build_wan_video` — CLIPLoader(wan/umt5) → 双 UNETLoader(高/低噪,fp8) → ModelSamplingSD3(shift=5) → WanImageToVideo → 双 KSamplerAdvanced(cfg=1,euler/simple) → VAEDecode → VHS_VideoCombine(h264) 落盘
